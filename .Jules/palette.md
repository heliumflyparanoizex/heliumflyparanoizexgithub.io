## 2025-02-18 - Essential Skip Links for Single-Page Applications
**Learning:** For single-page applications with heavy top navigation (like this one, which features a fixed header, a preloader, and numerous navigation items), users relying on keyboard navigation face significant friction tabbing through the entire menu before reaching the main content. This is a common accessibility issue in single-page apps.
**Action:** Always include a "Skip to main content" link immediately after the `<body>` tag, pointing to the primary content area (e.g., `#inicio`). Implement this using Tailwind utility classes (`sr-only focus:not-sr-only focus:absolute z-[9999]`) to ensure it remains visually hidden until explicitly focused via keyboard navigation, minimizing visual impact while maximizing accessibility.
